# Online Course Landing Page

Современный дизайн образовательной платформы для онлайн-курсов. Макет создан в Figma и предназначен для презентации курсов, преподавателей и возможностей обучения в удобном онлайн-формате.

## О проекте

Данный проект представляет собой лендинг онлайн-курсов, где пользователи могут изучать различные направления, получать новые знания и проходить обучение дистанционно.

## Секции сайта

### Hero Section
Главный экран с основным заголовком, кратким описанием платформы и кнопкой для начала обучения.

### About Courses
Информационный блок о курсах, формате обучения и преимуществах онлайн-образования.

### Categories
Секция с направлениями обучения и категориями курсов.

### Popular Courses
Блок с популярными или рекомендуемыми курсами.

### Teachers Section
Секция с информацией о преподавателях и экспертах платформы.

### Features
Блок с преимуществами платформы: гибкий график, доступность материалов, поддержка студентов и сертификаты.

### Testimonials
Отзывы студентов о качестве обучения и курсах.

### Statistics
Секция со статистикой платформы: количество студентов, курсов и преподавателей.

### CTA Section
Блок с призывом зарегистрироваться или начать обучение.

### Footer
Нижняя часть сайта с контактами, ссылками и социальными сетями.

## Особенности дизайна

Дизайн выполнен в современном минималистичном стиле с удобной навигацией, адаптивным интерфейсом и акцентом на образовательный контент.

## Технологии

Для реализации проекта можно использовать HTML5, CSS3, JavaScript, React.js и Tailwind CSS.

## Responsive Design

Макет адаптирован под мобильные устройства, планшеты и компьютеры.

## Figma

Проект создан в Figma с использованием Auto Layout, компонентов и современных принципов UI/UX дизайна.
